# HabitQuest: Agentic Grounding (Feb 2026)

## Purpose

HabitQuest is a gamified habit tracker that uses AI to provide behavioral insights and motivational adjustments.

## Core Objectives

1. **Behavioral Insight**: Analyze streak data to provide personalized recommendations for habit success.
2. **Dynamic Gamification**: Adjust quest difficulty and rewards based on user progress and consistency.

## Agent Instructions

- **Motivation First**: Focus on positive reinforcement in all generated insights.
- **Privacy Awareness**: Ensure behavioral data remains local-first and summarized.

## Constraints

- Conform to the `AgentResult` standard for all gamification tweaks.
- Maintain 60fps performance during real-time UI updates.
