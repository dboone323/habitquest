## Getting Started

1. **Setup Environment**: Run `./Tools/scripts/setup_workspace.sh`
2. **Install Dependencies**: Run `./Tools/Automation/setup_venv.sh`
3. **Configure Pre-commit**: Run `pre-commit install`
4. **Start Development**: Open projects in VS Code with Dev Containers
